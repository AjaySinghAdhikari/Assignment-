# 📚 Python & ML Assignments

A collection of hands-on assignments covering Python fundamentals, 
OOP, data libraries, and machine learning concepts.

## 📂 Topics Covered
- Python Basics (comments, functions, data types)
- Object-Oriented Programming (classes, abstraction, inheritance)
- Exception Handling
- Data Libraries: NumPy, Pandas, Matplotlib, Seaborn
- Machine Learning: Regression models
- Advanced Python: Multithreading, Multiprocessing, MySQL

## 🛠️ Requirements
- Python 3.x
- Jupyter Notebook
- Libraries: numpy, pandas, matplotlib, seaborn, sklearn, mysql-connector-python

## 🚀 How to Run
1. Clone the repo: `git clone <repo-url>`
2. Install dependencies: `pip install -r requirements.txt`
3. Open any notebook: `jupyter notebook`
